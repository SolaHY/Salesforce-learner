# SF Quest — Salesforce 認定アドミン 冒険の書

Salesforce Certified Administrator 資格の合格を、**RPG風クエスト感覚**で楽しく目指す学習ポータルです。
React + Vite で構築されており、冒険の記録はブラウザの localStorage に保存されます（サーバー不要）。

## ゲーミフィケーション

- **🧙 レベル & 称号**: XPを貯めてレベルアップ。「見習いトレイルブレイザー」から「伝説のアドミン」まで8段階の称号。
- **🔥 連続学習ストリーク**: 毎日学習して連続日数を伸ばそう。
- **⚡ コンボ**: クイズで連続正解するとコンボが加算。
- **🏅 バッジ（称号コレクション）**: 「初陣」「連続正解5」「賢者」など8種の実績を解放。
- **🎉 演出**: レベルアップ・バッジ獲得トースト、クリア時の紙吹雪。

## 機能

- **⚔️ クエスト（問題演習）**: 出題範囲別 / 全範囲の4択・複数選択問題。即時採点・解説・S〜Dランク評価。
- **🃏 記憶の修練場（フラッシュカード）**: 用語暗記のフリップカード。習得で +5 XP。
- **📖 知恵の書庫（学習教材）**: 各ドメインの要点まとめ。
- **🏆 冒険の記録（進捗）**: レベル・正答率・最大コンボ・バッジ・ステージ攻略度・戦いの記録を可視化。

## セットアップ

```bash
npm install
npm run dev      # 開発サーバー (http://localhost:5173)
npm run build    # 本番ビルド (dist/)
npm run preview  # ビルド結果のプレビュー
```

## コンテンツの追加・編集

サンプルコンテンツは `src/data/` にまとまっています。JSON的なJSオブジェクトを編集するだけで増やせます。

- `src/data/domains.js` — 出題範囲の定義
- `src/data/quizzes.js` — 問題（`answer` は選択肢のインデックス。複数正解は `multi: true` + 配列）
- `src/data/flashcards.js` — フラッシュカード（表 `front` / 裏 `back`）
- `src/data/studyMaterials.js` — 学習教材のまとめ

> ⚠️ 同梱のコンテンツは学習用のサンプルです。最新の試験範囲は公式の
> [Salesforce 認定アドミニストレーター試験ガイド](https://trailhead.salesforce.com/credentials/administrator)
> を確認してください。
